# Meu Diario Virtual
O Meu Diário Virtual é um aplicativo de diário pessoal com visual delicado em tons de rosa, inspirado em aplicativos de notas e em um caderno tradicional. Ao abrir o aplicativo, o usuário encontra uma tela inicial com o nome "Meu Diário Virtual" em estilo lettering preto e um botão "Entrar". Para acessar o diário, é necessário realizar um cadastro informando um nome de usuário e criando uma senha com no mínimo 12 caracteres. Após o cadastro, o usuário é direcionado para o diário, que possui aparência de caderno, permitindo escrever livremente suas anotações, pensamentos, memórias e compromissos. Sempre que desejar iniciar um novo registro, pode utilizar o botão "Nova Página", que cria uma folha em branco adicional. Todas as páginas são salvas automaticamente no navegador enquanto o usuário escreve, garantindo que as anotações não sejam perdidas. O aplicativo também conta com um botão "Sair", que encerra a sessão e retorna à tela inicial, exigindo que o usuário faça login novamente para acessar o conteúdo do diário. O objetivo do Meu Diário Virtual é oferecer um espaço simples, organizado e acolhedor para registrar momentos, ideias e lembranças de forma prática e segura. 🌸📖✨
